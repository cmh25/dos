# 20230124
Built in dosbox 0.74-3 with masm 6.11
To compile, just: ml vga00x.asm

vga001.asm      - plots a blue dot in the middle of the 
                  screen

vga002.asm      - bit mask demo

vga003.asm      - dotted line in the middle of the
                  screen

vga004.asm      - fills screen with blue dots

vga005.asm      - demonstrates 8x8 fonts in mode 12h

vga006.asm      - displays the 16 colors of mode 12h

vga007.asm      - displays the 256 colors of mode 13h
